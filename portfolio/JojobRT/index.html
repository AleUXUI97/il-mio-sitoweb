JojobRT

The Project

The project initially involved a general brand analysis, including the definition of vision and mission, and an evaluation of the current website through an AS-IS analysis (10 usability heuristics by J. Nielsen).
To identify the key characteristics the website should include, in addition to creating a questionnaire and defining the target audience it addresses, I conducted a competitor analysis of JojobRT using benchmarking tables.
Following these analyses, I redesigned the website’s UI and subsequently conducted a user testing phase, which is currently still in progress.
